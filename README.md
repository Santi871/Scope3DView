# Scope 3D View

This plugin for [N.I.N.A.](https://nighttime-imaging.eu/) adds a real-time dockable 3D view of the telescope within the application.

This plugin is based on [GSPoint3D](https://sites.google.com/view/greenswamp/gspoint3d). Check them out!
